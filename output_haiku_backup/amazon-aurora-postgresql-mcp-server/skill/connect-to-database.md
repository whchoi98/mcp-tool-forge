# Connect To Database

Connect to a specific database and save the connection internally

## Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `region` | string | Yes | region |
| `database_type` | string | Yes | database type |
| `connection_method` | string | Yes | connection method |
| `cluster_identifier` | string | Yes | cluster identifier |
| `db_endpoint` | string | Yes | database endpoint |
| `port` | integer | Yes | Postgres port |
| `database` | string | Yes | database name |
